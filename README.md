# 阿斌智能协同云图库 🌟
一款面向个人/企业的一站式智能云图库解决方案，支持公共素材共享、个人私有空间、团队实时协同编辑，基于主流前后端技术栈构建，兼顾高性能与易用性。

## 📖 项目介绍
### 核心定位
阿斌智能协同云图库聚焦「图片资产管理」核心场景，解决个人/企业图片分散存储、协作效率低、检索困难等痛点，提供：
- 📸 **多场景存储**：公共图库（素材共享）、个人私有空间（网盘/相册）、企业团队空间（协同管理）
- 🚀 **智能能力**：AI 图片分析、智能标签生成、多维检索（关键词/标签/格式）
- 👥 **团队协同**：基于 WebSocket 的实时协同编辑、细粒度权限管控、操作日志审计
- ⚡ **高性能**：多级缓存、分库分表、异步处理，支撑高并发图片上传/检索

### 应用场景
- 个人：私人相册、作品集管理、素材收藏
- 企业：团队活动相册、设计素材库、营销图片管理
- 公共：表情包/壁纸/设计素材共享平台

## 🛠 技术栈
### 前端技术栈
| 技术/框架         | 版本/用途                          |
|-------------------|------------------------------------|
| Vue 3             | 核心前端框架（Composition API）    |
| TypeScript        | 静态类型检查，提升代码可维护性     |
| Vite              | 高性能前端构建工具                 |
| Ant Design Vue    | 企业级UI组件库                     |
| Pinia             | 轻量级状态管理                     |
| Axios             | 网络请求封装（拦截器/统一错误处理）|
| WebSocket         | 实时协同通信                       |
| ECharts           | 数据可视化（图库统计/分析看板）    |

### 后端技术栈
| 技术/框架         | 版本/用途                          |
|-------------------|------------------------------------|
| Spring Boot       | 后端核心框架                       |
| MySQL + MyBatis-Plus | 关系型数据库 + ORM框架            |
| Redis + Caffeine  | 多级缓存（本地+分布式）            |
| 腾讯云 COS        | 对象存储（图片文件存储）           |
| ShardingSphere    | 分库分表（海量图片元数据存储）     |
| Sa-Token          | 权限控制（RBAC模型/空间权限）      |
| Disruptor         | 无锁异步队列（高并发图片上传）     |
| DDD 架构          | 领域驱动设计（业务解耦）           |
| 阿里云 AI API     | 图片智能分析/标签生成              |

## ✨ 核心功能
### 1. 公共图库模块
- 图片多方式上传（文件/URL）、批量操作
- 管理员审核机制（待审核/通过/驳回）
- 热门标签/分类检索、访问量统计分析

### 2. 个人私有空间
- 图片私有化存储、自定义分类/标签
- 在线图片编辑（裁剪/调色/加水印）
- 扫码/链接分享（权限可控，支持有效期）

### 3. 企业团队空间
- 团队创建/成员邀请/角色权限配置
- 多端实时协同编辑（操作实时同步）
- 团队资源共享、版本回溯、操作日志

### 4. 系统管理
- 用户管理（注册/登录/权限配置）
- 全局数据看板（存储/流量/访问统计）
- 系统配置（存储策略/缓存规则/限流配置）

## 🚀 快速启动
### 前端启动
```bash
# 克隆仓库
git clone https://github.com/small1zhang/binpicture.git
cd binpicture/frontend

# 安装依赖
npm install

# 开发环境启动
npm run dev

# 生产环境构建
npm run build
```

### 后端启动
```bash
# 进入后端目录
cd binpicture/backend

# 配置环境（修改application.yml中的数据库/Redis/COS信息）
# 编译打包
mvn clean package

# 启动服务
java -jar target/abin-picture-1.0.0.jar
```

## 🎯 技术亮点
1. **高性能缓存设计**：Caffeine 本地缓存 + Redis 分布式缓存，降低数据库压力，检索性能提升 80%+
2. **海量数据支撑**：ShardingSphere 分库分表，支持千万级图片元数据存储
3. **实时协同能力**：WebSocket 实现多端状态同步，延迟 < 100ms
4. **安全权限管控**：基于 RBAC 模型的细粒度权限，支持空间/图片级别的权限校验
5. **AI 增强能力**：接入阿里云 AI API，实现图片智能分析、标签自动生成
